
## スプラウト.rb（sprout.rb）とは

Ruby on Railsを仲間と一緒に学びたい人のための勉強会(読書会) です。

この勉強会の最大の目的は、継続的に学ぶことです。

セミナーのように受講するというスタイルではなく

仲間と「Ruby on Rails チュートリアル」を読み進めながらワークをしていきます。
